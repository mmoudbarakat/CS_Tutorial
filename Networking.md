## IPv4 Address Structure

- Example: 192.168.1.1
- Total size: 32 bits
- Split into 4 sections (octets)
- Each octet = 8 bits
- 8 bits → 2^8 = 256 values (0–255)

### Binary Form
192.168.1.1 =
11000000 10101000 00000001 00000001

### Important
- Dots (.) are only for human readability
- In reality, IPv4 is just 32 continuous bits
